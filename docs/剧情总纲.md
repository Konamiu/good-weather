# 《好天气》剧情总纲（真实故事档案）

> 本文档是全游戏的事实底本：所有章节、台词、素材都从这里取材。
> 来源：东的口述 + 三封手写信（2021.5.20 / 6.1 / 6.21，照片见 `public/assets/real/letters/`）。
> 规则：**只用真事，不虚构情节**。空白处标注【待补】。

## 人物

- **东**：男主，作者本人。单亲家庭长大，缺安全感，敏感、内向、不爱说话，被她说"高冷"（其实是心像被冰封住）。用"记录"来爱人：数日子、写信、记备忘录。信件署名"东"。
- **楠**：女主。18岁跟了他。齐耳短发，很漂亮，懂事、细心、听劝。笑容"像蜜枣一样甜"。唱歌好听，唱过《山楂树之恋》。信件抬头"亲爱的楠"。
- 她妈妈（第6章出场）、他姐姐（瑞安，第7章提及）。

## 完整时间线

### 第一段（2019夏 → 2019.8）

- **2019年夏**，他高中毕业，在市里打暑假工，**QQ上认识**。他发传单——不是街头递，是**整栋楼、整个小区、老小区地爬楼，把传单塞到每户门口**；她做游戏客服。
- 她每天嘘寒问暖：**多喝水、拿湿巾擦汗、别中暑**。他"内心坚硬的冰逐渐被这一阵阵的温暖融化"。她说他声音好听——其实他故意用普通话压低声音。
- 一句 **"试试吧"**，确定关系。
- 后来他去温州牛排店，在**水吧切水果做饮料**（约9:00-12:00 / 14:00-19:00），她已离职。异地。
- 他怕失去："我如果要谈，就想谈一段很久很久的恋爱。"
- **话题聊完危机 → 备忘录方法**：每天把身边有趣的事记进手机备忘录，下班组织好语言打电话讲给她听（这个习惯保持到今天）。她认真听，也分享，还**每天唱他最爱听的歌**。他一下班能收到**几十条消息**。"我真的很开心，我真的很幸福，那是我最难忘的一段记忆。"
- **2019年8月中下旬**：她前男友回来找她，她提出分手。而他几天前刚收到她送的礼物。**他一气之下骂了她。**"自此好像失了灵魂一样，我不再像以前那样开朗、闪闪发光、开心了。"

### 空窗两年（2019.8 → 2021.2）

- 疫情时期。他郁郁不得志，更加谨慎敏感，**拒绝了好几段感情**。
- 两人偶尔互相看动态、看对方在听的歌；她发朋友圈，他会回应。
- 大学期间，因她说过 **"我喜欢床铺干净整洁的男孩子"**，他每天早晨把床铺整理干净再出门；晚上翻她的照片、听她唱过的歌，"心里仿佛得到了安慰，期盼着你能早日回来"。

### 第二段（2021.2 → ？）

- **2021.2.9**（过年前后）她回来了，说：**"不会让你白等的。"** 复合。
- **第一次见面：阜阳火车站**，她先来找他。
- 约定**轮流奔现**："第一周她来找我，后一周我去找她。"
- **他去亳州**：到她家附近**被她妈妈看到**，她跟家里吵了一架；两人快上公交去市里时她被家里人拉走——**他一个人坐上了那趟开往市里的公交**。也是那次，他**第一次收到女生送的鲜花**。信中"凌晨的烤冷面、一起看电影、打伞陪她去医院"都发生在亳州。
- **2021.5.20信**：相识699天、相恋101天。"不得不说，有你真好！"
- **2021.6.1信**（儿童节）：因学校原因27天没见。"每天只能看到你十几分钟，但已经足够。""终于体会到那种迫切等你下班的感觉。"期待"一起上下班、对你说晚安、给你盖被子、捶腰捶腿、讲故事、暖肚子暖被窝暖脚脚"。**"我楠楠小朋友，我会像保护眼睛一样保护你！"**
- **2021.6.21信**（相识两周年）："我是奔着结婚去的！""你在最美的年纪跟着我，我可不能让你受伤害和委屈。"
- **2021五一**：他请假去亳州找她。
- **2021暑假：瑞安同居一个多月**。她先去瑞安找暑假工（**吾悦广场**上班）、租好房子。他到瑞安那天下小雨，钥匙在她那，他在**网吧躲雨**；等她快下班时**推着行李箱跑上商场扶梯**去找她——相遇。
  同居日常：他学前端、**给她洗衣服**、带她去也在瑞安的姐姐家、逛商场、吃牛排（"贼好吃"）、**无论多大雨都接送她上下班**——有次大雨他打伞去接，半路看到她，"心疼坏了"；有一次**两人碰巧哼着同一首歌**。
- 暑假结束：**一起坐火车回阜阳，一起回家**。
