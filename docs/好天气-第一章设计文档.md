# 《好天气》第0章+第1章 详细设计

## 第0章 · 备忘录（游戏的壳，每章入口）

**画面**：黑屏 → 一部像素手机亮起（已有 ui_phone_frame）→ 备忘录APP列表。
**列表内容**：按日期排列的条目，通关进度决定解锁。未解锁条目显示灰色"？？？"。第一条：`2019.6 · 蝉鸣`。
**交互**：点条目进章节；**长按已通关条目 = 显影**（浮现真实素材：聊天截图/照片，本期占位）。
**细节**：列表里混 1-2 条"不是关卡"的条目，点开只是一段备忘录原文（后续放东的真实备忘录文字）。

## 第1章 · 蝉鸣（2019夏 · 发传单）

### 场景
夏日小城街道（已有 bg_summer_street），正午强光。章节后段整体调色渐变为黄昏（代码色调叠加，不需新图）。

### 人物
- 男主"东"：发传单打工
- 路人 ×3款（男女老少，跑动/摇头动画）
- 狗 ×1（纯搞笑担当，不接传单，会叼走一张）
- "楠"：**本章不出场**，只存在于手机里（QQ消息）——她是游戏客服，两人刚在QQ认识

### 剧情节拍
1. 旁白卡：`2019年夏天，高中毕业。我在市里打暑假工，发传单。`
2. **玩法A 发传单（爬楼版，2026-07-14修正）**：真实情况是整栋楼、整个小区地跑，把传单塞到每户门口。楼道竖向剖面图，点门走过去塞传单（门缝留下白色传单），点楼梯爬上一层，一层层塞完整栋楼（6层×2户）。彩蛋反应：门后狗叫"汪汪汪！"、住户"谁呀？！"。竖屏手机天然适合往上爬
3. 每塞3张左右，手机震动 → 她的消息弹出（嘘寒问暖：多喝水/湿巾擦汗/别中暑）——**楼爬得越累，她的消息越暖**，这是本章的情绪结构
4. 8张完成 → 黄昏 → 玩法B 打字回复：她问"今天累不累"，二选一回复，怎么选她都接得很暖，末条："明天也给你发消息呀"
5. 章末卡（打字机逐字）：`那年夏天，我们聊了很多很多。后来我说：` → 大字 `「试试吧」` → 备忘录解锁下一条目动画

### 交互清单
| 操作 | 行为 |
|---|---|
| ←→ / 点击屏幕左右半区 | 移动 |
| 空格 / 点击男主 | 递传单（附近有路人时才生效） |
| 长按弹出的消息 | 显影真实聊天截图（占位） |

### BGM / 音效
- 底噪：**蝉鸣**（白噪声带通+颤音，代码合成）
- BGM：轻快8-bit小调循环，黄昏后减速变柔
- 音效：消息"滴滴"两连音（QQ风）、递出"嗖"、拒绝"噗"、接受上扬三音
- 本章不用主题曲——《山楂树之恋》动机留到她第一次唱歌的章节才出现
