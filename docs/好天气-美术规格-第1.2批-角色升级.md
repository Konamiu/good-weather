# 《好天气》美术规格 — 第1.2批（角色质感升级）

给Codex：1.1批角色整体作废。两个根本改动：**帧升级到 48×64**（脸部像素翻倍才画得出五官），**上色从平涂升级为三阶明暗**。目标质感对标开罗(Kairosoft)《口袋学院》人物——不是复制它的图，是复制它的工艺。

## 人设修正
- **楠：齐耳短发**（不是长发！）+ 刘海 + 暖黄发卡。漂亮、爱笑。米白上衣+红裙保留
- **东：帅气清爽**。黑短发（额前碎发），蓝短袖深长裤保留

## 工艺规则（开罗质感的核心，逐条执行）
1. **三阶上色**：每个主色区（头发/衣服/裙子/皮肤）必须有 底色+暗部+亮部 三阶。光源固定左上方：右下留暗部，左上顶部留1-2px亮部
2. **眼睛**（好看与否80%在这）：宽2px×高3px，深棕底 + **左上1px白色高光点**。楠的眼睛比东高1px（更大更亮）。眼距5-6px
3. **嘴巴**：必须有。1-2px暗红 `#a6524a`，默认微微上弯（两人都是爱笑的人）
4. **鼻子**：1px皮肤暗色点，位于两眼中点下方2px（若显脏可省，嘴不可省）
5. **腮红**：每侧1×2px `#e8a598`，位于眼睛正下方2px、偏外侧
6. **下巴/脖子交界**处1px皮肤暗色阴影
7. **描边**：废除纯黑一圈，改**选择性描边(selout)**——轮廓色取相邻区域主色的最深阶（头发外缘用发色深阶，皮肤外缘用 `#8f5f4a` 类肤色深阶），只有最外轮廓描，内部区域交界靠明暗区分
8. **发型是辨识度的一切**：楠的短发要画出"内扣弧度"+刘海分缕（2-3缕），东的碎发2-3撮。头发暗部在发梢内侧，亮部一条在头顶偏左
9. 底部阴影椭圆保留（`#00000040`）

## 色板（在第1批16色基础上新增）
- 亮阶：发亮 `#5a4a56`、蓝亮 `#6b8fc2`、红裙亮 `#d97d70`、米白亮 `#fffdf6`
- 暗阶：肤暗 `#c9926b`、蓝暗 `#33507c`(已有)、红裙暗 `#8f3d38`、米白暗 `#d8cdb8`
- 五官：嘴 `#a6524a`、腮红 `#e8a598`、眼白高光 `#ffffff`
