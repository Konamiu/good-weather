# 批次B2 · 第1章楼道场景扩充（给Codex直接作图）

> 目标：把楼道从"几何占位"变成**有生活痕迹的2019年中国老小区**。
> 参考质感：开罗游戏的密度 + 《中国式家长》《完美的一天》里的市井感。
> 使用方法：先贴《美术提示词全集.md》开头的【全局风格块】，再贴本文档具体条目。
> 拆分原则：墙面基底 + 墙面贴片 + 地面道具 + 门变体 + 活物动画 分开交付——引擎逐层随机组合，6层每层不重样。

## 1. 墙面基底（3款可平铺层段）

`wall_base_3s_360x90.png`（三款横排拼一张，每款360×90）

每款都是一个楼层段的墙面底子：上2/3刷白发黄墙面（#cfc6b0，带轻微色斑变化，不要纯色平涂——加入3-4处更浅/更深的大块渐变斑），下1/3墨绿色墙裙（#5f7355底，漆面剥落处露出水泥灰#8a8580，每款剥落位置不同）。三款区别：
- 款A 较新：墙裙完整，仅1处小剥落
- 款B 普通：2-3处剥落，墙面一道竖向水渍
- 款C 破旧：大片剥落、底部返潮发黑（#4a4a44晕染）、一道裂缝斜穿

## 7. 引擎侧组合规则（给Claude，Codex不用管）

- 每层随机：墙面基底1款 + 墙贴2-4件 + 地面道具1-3件 + 门款式随机（同层两户不同款）
- 固定编排：1F=信报箱+电瓶车+大爷摇扇+狗窝；顶层最空（只有天光）——越往上越安静
- 猫在3F窗台打盹，玩家靠近会站起来走两步
- 大妈/送水工/小孩作为随机楼道事件：错身时男主自动侧身让路（塞传单的节奏被打断一拍，生活感）
- 声控灯：男主经过时"啪"亮起（配音效），走远2秒后灭
- 窗外风景按层替换，光斑颜色随层数从暖黄渐变到亮白

## 验收标准

1. 沿用全集的管线标准（等距/透明/锐利/色板）
2. **组合测试**：随机抽3件墙贴+2件道具+1款门拼在墙面基底上，必须和谐不打架（同一世界的东西）
3. 所有道具在1x原尺寸下可辨认（不放大也知道那是蜂窝煤/白菜/电瓶车）
