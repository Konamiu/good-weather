# Ninja Adventure 像素工艺研究

参考来源：Pixel-Boy / AAA 的 [Ninja Adventure Asset Pack](https://pixel-boy.itch.io/ninja-adventure-asset-pack)，CC0 1.0。研究副本仅临时下载到 `/tmp`，不进入本仓库。

用于《好天气》1.2角色升级的结论：

1. 亮部不是平均铺开，而是集中在左上轮廓和头顶小色块；右下保留连续暗面，角色在1x下才有体积。
2. 外轮廓不应统一纯黑。头发、皮肤、衣服分别使用本区域最深阶，亮侧轮廓允许变浅，暗侧保持最深。
3. 眼睛高光只需要1px，并必须落在眼睛左上；高光周围保留足够深色，才能在原始尺寸读成“有神”。
4. 内部区域主要靠明暗交界分隔，不给每个零件都包黑边；否则放大后像拼装色块。
5. 动画帧优先保持头身母版一致，只在脚、手、重心和发梢处变化，避免人物在循环中变形。

本项目只借鉴上述工艺和CC0脸部像素组织，不复制角色整图、服装或造型。
