# 第1章重构 · 俯视探索版（调研 + 新设计）

## 一、三款参考作的钩子拆解

### 口袋妖怪（GBA世代）
1. **开局10分钟给两层目标**：长目标（成为冠军/集齐图鉴）+ 永远可见的短目标（下一个道馆在哪）。玩家任何时刻都知道"我现在去哪、为了什么"
2. **探索即奖励**：草丛、可调查的告示牌、每个NPC都有一句话、隐藏道具——世界有"密度"，走到哪都有反馈
3. **图鉴=收集欲**：空格子本身就是钩子
4. **视角**：俯视3/4视角（能看到建筑正面+屋顶），世界感强

### 星露谷物语
1. **日循环**：早晨计划→白天行动→晚上结算睡觉。"再玩一天就睡"的魔力全在这个节奏
2. **好感度系统**：对话/送礼→心形格子可视化增长→解锁新剧情事件
3. **延迟满足**：今天播种，四天后收获

### 去月球
1. **倒叙悬念**：先看到结果，不知道原因
2. **记忆碎片拼图**：收集物承载叙事
3. **主题曲变奏**（已在我们的音乐方案里）

## 二、新第1章设计《蝉鸣》

**视角**：俯视3/4视角小镇（口袋妖怪式）。**进入单元楼时切到已做好的横版楼道**（口袋妖怪进建筑也是切地图，我们的爬楼玩法完整保留，一点不浪费）。

### 钩子系统（三层，全部来自真实故事）

1. **章节大钩子——「冰封的心」**：屏幕角落一颗冰封的像素心（他信里写自己"心像被冰封住"）。她每一次嘘寒问暖、每一晚聊天，冰裂开一点。**冰完全融化=说出"试试吧"=章节通关。**玩家永远看得到进度。
2. **日循环**：白天=发传单指标（去指定小区跑楼，横版玩法）+自由探索小镇；晚上=回出租屋和她QQ夜聊。共4天，每天有新内容。
3. **趣事图鉴（备忘录）**：小镇里散布可发现的「趣事」（猫爬树下不来、大爷下棋悔棋吵架、西瓜摊主睡着了……）。发现→记进备忘录→**晚上聊天时变成对话选项**：收集越多，晚上跟她越有得聊，冰化得越快。这就是他真实的"备忘录方法"的游戏化——**收集趣事=为她收集一整天**。

### 小镇地图（俯视，约3屏×3屏）

- **出租屋**（存档点/夜聊触发点）
- **老小区A/B/C**（发传单任务点，进楼切横版楼道）
- **街心**：西瓜摊、烤冷面摊（暂关着——第6章才吃，埋伏笔）、小卖部
- **网吧**（伏笔：第7章瑞安网吧躲雨会呼应）
- **树荫棋摊、健身角、传达室**（NPC密度点）

### 4天节拍表

- **D1**：入职教学→跑1栋楼→晚上她的第一条消息（冰裂第一道缝）
- **D2**：指标加倍+狗事件；解锁趣事图鉴（她说"今天有什么好玩的呀"→玩家意识到要收集）
- **D3**：高温天，她连发"多喝水/湿巾擦汗"（信里原话）；帮遛狗大妈找狗支线
- **D4**：最后一栋楼+全镇趣事收尾→晚上长对话→冰心完全融化→黑屏→**「试试吧」**→章末卡

### ⚠️ 真实性边界（必须遵守）

小镇NPC和趣事是**游戏化填充**（真实趣事内容已不可考），但"备忘录收集趣事讲给她听"这个机制100%真实。**东：如果你还记得当年备忘录里的真实趣事，发给我，我逐条替换掉编的。**她的消息全部用信件原话，不新编。

## 三、画风方向修正

用户不喜欢当前画风。修正方向：
- **视角**：俯视3/4（口袋妖怪绿宝石/星露谷同款），不再是横版剪面
- **色彩**：饱和度提高一档，夏天要"晒"——参考星露谷夏季的浓绿+GBA宝可梦的明快
- **密度**：地图上要"满"——树影、电线杆、草丛、井盖、长椅，拒绝大片空地
- 横版楼道作为室内子场景保留（进楼切场景），与俯视世界用同一色板
- 角色沿用1.2批48×64（俯视世界里同样成立），四方向已齐
