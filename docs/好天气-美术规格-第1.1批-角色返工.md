# 《好天气》美术规格 — 第1.1批（角色返工）

给Codex：第1批的**背景、手机框、雨幕验收通过，保留**。5个角色文件全部作废重做。管线（代码生成、横条雪碧图、16色板、命名规则、无抗锯齿）不变，沿用第1批全局要求。

## 返工原因
侧视无脸方块人没有情绪表达能力。改为 **RPG 俯视Q版**（参考《宝可梦火红叶绿》《星露谷》NPC 比例），正面为主，必须有脸。

## 角色新比例（帧仍为 32×48）
- **头占高度约40%**：头部（含发型）约 18-20px 高、宽 16-18px；身体+腿约 24px
- 角色实际轮廓高约 42-44px，底部留 2px 阴影椭圆（#00000040）
- **脸**：正面时必须有——眼睛为 2×2px 深棕色两点（间距 4-6px），眼下各 1px 腮红 `#e8a598`（新增色板允许色）。不画嘴和鼻（需要表情时后续批次出专用帧）
- 朝上（背面）：只画后脑发型，无脸
- 侧面：单眼 2×2px，鼻尖 1px 外凸可选

## 造型设定
- **男主"东"**：黑色短发（额前2-3撮碎发像素），蓝色短袖 `#4a6fa5`，深色长裤，白色鞋。气质干净清瘦
- **女主"楠"**：黑色及肩长发（正面两侧垂下、背面呈整片发帘），米白上衣 `#efe6d5` + 红裙 `#c4554d`，比男主整体矮 4px，头发上可加 1 个暖黄发卡 `#f2cc8f`（她的辨识点）
