# 第1章 · 地图与开场重构

> 解决三个问题：地图空、开场糙（一上来就上班）、建筑进不去。

## 一、开场重构：加一个「序幕日 D0」，不上班

宝可梦不会让你开局打道馆，星露谷第一天也只是收拾农场。第1章改为 **D0序幕 + D1-D4 工作日**：

### D0 · 到达（约6分钟，无任何指标）

1. **开场镜头**：长途客车站牌前，东拖着行李箱下车（脚下是陌生的小城）。旁白："2019年6月，高考完的那个夏天。我一个人来市里，打人生第一份工。"
2. **找到出租屋**（唯一目标，地图有指引箭头）：进屋→放下行李箱→屋里可以逐件调查（床/旧风扇/日历/空桌子——每件一句内心独白，建立"一个人"的底色）
3. **自由探索**：镇上转一圈，此时所有NPC都是"陌生"级台词（传达室大爷拦你盘问、小卖部老板娘搭话、棋摊大爷懒得理你）——**孤独感是玩出来的，不是台词说的**
4. 路过雇主的门面：贴着"招传单派发员"→揭下来（明天的工作，钩子埋下）
5. **晚上回屋**：躺床上无聊翻手机→QQ上遇到她〔相识具体方式待东补充，现用"群里聊起来"占位〕→第一次对话（她的第一句嘘寒问暖）→**冰封的心UI首次出现，全封冻状态**→睡觉，D1开始
6. 设计意图：先让玩家拥有"我的屋子、我的小镇"，再开始日循环——她的出现才有"照亮"可言

### D1-D4 维持原日循环设计（发传单+探索+夜聊），不再赘述

## 二、地图扩容：从20×22到 48×40（1536×1280，约4屏×2屏）

五个分区，每区有明确的"性格"和玩法职责：

```
┌─────────────┬───────────────┬──────────────┐
│ 西北·老小区群   │  北·出租屋巷      │ 东北·河与桥     │
│ 单元楼A/B/C围合 │  出租屋+两户平房邻居│ 小河+石桥+柳树   │
│ 传达室+大门+晾衣 │  巷子窄、有台阶    │ 钓鱼大爷〔填充〕  │
├─────────────┴──┬────────────┴─┬────────────┤
│ 中·主街（横贯东西） │ 街心十字        │  (主街东段)     │
│ 电线杆+梧桐行道树   │ 小卖部/网吧      │  客车站牌(D0起点)│
│                 │ 烤冷面摊(歇业)/西瓜摊│              │
├────────────────┼──────────────┴────────────┤
│ 西南·小广场        │ 东南·空地与半拆的老墙            │
│ 棋摊/宣传栏/健身角  │ 杂草+砖堆——小城的"没建完"感      │
│ 乒乓球台/滑梯      │ 金趣事点：墙洞里的猫窝〔填充〕      │
└────────────────┴───────────────────────────┘
```

- **老小区群**：三栋楼围合出真正的"院子"（晾衣绳/花坛/自行车棚在院内），发传单目标全在这区——工作区
- **出租屋巷**：他的领地。窄巷+台阶制造"回家"的空间记忆
- **街心**：消费与NPC密度最高点——生活区
- **河与桥**：纯情绪空间，没有任何任务。傍晚在桥上能看到夕阳（金趣事点）——**留白区，给"想她"用**
- **东南空地**：小城的粗糙真实感，破而不脏

道路层级：主街（水泥3格宽）→ 小区路（水泥2格）→ 巷子（土路1-2格）→ 草地小径（磨损叠加）。

## 三、可进入建筑（第1章5个室内）

| 建筑 | 室内内容 | 玩法 |
|---|---|---|
| **出租屋** | 床/桌/风扇/日历/行李箱/充电的手机 | 存档点+夜聊触发点+日历彩蛋，每天回来睡觉 |
| **单元楼A/B/C** | 已有的横版爬楼玩法 | 发传单（进楼切场景，已实现） |
| **小卖部** | 货架/冰柜/柜台老板娘 | 买水买雪糕（水分系统的补给站） |
| **传达室** | 大爷的窝：收音机/茶缸/报纸/小风扇 | 好感解锁后才让进，进去=蹭凉+听评书（金趣事） |
| 网吧 | **进不去**（"没什么进去的理由"）| 跨章伏笔，第7章呼应 |

室内实现：每个室内是独立小地图（8×7格左右），门口踩触发点切场景，出门回到门前。

## 四、场景密度基准（治"空洞"）

依据environment-design规范定硬指标：
- 每屏（11×20格视野）至少：**1个焦点物 + 3-5件中型物件 + 8-12件小杂物/贴片**，同时保持30-50%安静地表
- 主街两侧电线杆每6格一根、行道树每5格一棵交错
- 所有墙根/栅栏根部必须有杂草或苔藓贴片过渡，禁止建筑"贴"在草地上
- 院内晾衣绳、竹竿、花盆按"有人住"的逻辑摆放（衣服晾在能晒到太阳的南侧）
