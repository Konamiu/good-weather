# 真实素材归档（显影系统的弹药库）

把真实素材文件按下面的结构放进来，引擎按目录约定加载。
**这个目录是整个游戏最珍贵的部分，git会跟踪它——不要放进 .gitignore。**

```
real/
├── letters/     手写信照片
│   ├── 2021-05-20_p1.jpg   (520信 第1页)
│   ├── 2021-05-20_p2.jpg
│   ├── 2021-06-01_p1.jpg   (儿童节信)
│   ├── 2021-06-01_p2.jpg
│   ├── 2021-06-21_p1.jpg   (两周年信)
│   └── 2021-06-21_p2.jpg
├── chats/       聊天记录截图（QQ/微信），命名：YYYY-MM-DD_主题.png
├── photos/      合照与地点照片，命名：YYYY-MM-DD_地点或事件.jpg
├── memos/       东的真实备忘录截图或文字导出（.txt也行）
└── audio/
    ├── her_song.mp3        她唱《山楂树之恋》的录音（找到后放这里）
    └── voice_*.mp3         其他语音
```

规则：
- 文件名里的日期尽量准确——显影时会显示在角落
- 图片不用预处理，引擎负责像素化第一眼+长按显影原图
- 涉及隐私（手机号/住址等）的截图先自行打码再放入
